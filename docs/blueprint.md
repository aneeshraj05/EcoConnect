# **App Name**: KanyaGuide

## Core Features:

- User Authentication: Secure user registration and login via email/password and Google, leveraging Firebase Authentication.
- Explore & Search: Intuitive search bar for destinations, dates, and budget, coupled with browsable categories and filters for tourist attractions.
- Accommodation & Experiences Listings: Display detailed listings of hotels, homestays, local guides, crafts, and cultural events with images, descriptions, pricing, and availability.
- Booking & Contact System: Enable users to book accommodations directly and contact local experience providers, with booking details stored in Firestore.
- User Dashboard: Personalized dashboard displaying user profile information, saved places, and a comprehensive booking history, powered by Firestore and Firebase Auth.
- AI-Powered Recommendations: Generate personalized attraction and experience recommendations based on user activity and preferences using an AI tool.

## Style Guidelines:

- Primary color: A vibrant, nature-inspired blue-green (#22A3C2) reflecting the ocean and sky, providing a fresh and inviting feel.
- Background color: A very light, subtle blue-green (#EAF6F9) that creates an airy, clean, and spacious canvas for content.
- Accent color: A lively, refreshing green (#60DBAC) for interactive elements and highlights, echoing lush greenery and natural beauty.
- Overall font: 'Inter' (sans-serif), chosen for its modern, clean lines, excellent readability across devices, and professional appeal for both headlines and body text.
- Utilize a consistent set of minimalist line icons for navigation and key functionalities, ensuring clarity and a clean aesthetic.
- Embrace a responsive, card-based design similar to Airbnb, featuring rounded corners and subtle soft shadows to create a friendly, engaging, and modern visual structure.
- Implement smooth transitions for page navigation and subtle hover effects on interactive elements and cards, enhancing the user experience with understated elegance.
